# perceptron-classify
Using a simple Perceptron to classify (linearly separate) some datapoints on a graph

## Setup
```
$ python3 -m venv venv
$ source ./venv/bin/activate
$ pip install -r requirements.txt
```

## Test
```
$ python3 Perceptron_Test.py
```

## Static Analysis
```
$ pycodestyle *.py
```
